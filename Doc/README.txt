These are Sandcastle Help File Builder's project files.

To get Sandcastle Help File Builder, see
http://shfb.codeplex.com/

To build the documents, you also need to install
MultilanguageDocumentationPlugIn from Poderosa project.
CVS repository: /tools/shfb/MultilanguageDocumentationPlugIn

To know how plugin should be installed,
please see "Special Folder Locations" section
on the Sandcastle Help File Builder Documentation.
